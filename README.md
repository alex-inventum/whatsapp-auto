# WhatsApp Auto

PWA para automatizar WhatsApp - Respuestas automáticas, clasificación y reenvío.

## Estructura

- `/frontend` - Next.js PWA (deploy en Vercel)
- `/backend` - open-wa server (deploy en Railway)

## Stack

- Next.js 14 + PWA
- @open-wa/wa-automate
- Supabase (PostgreSQL + Auth)
- Vercel (frontend)
- Railway (backend)
